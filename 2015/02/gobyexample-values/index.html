Go by Example － 值

20/Feb 2015
Development Go

Go语言中的值有各种类型，如strings，intergers，floats，booleans等，下面是一些简单的例子：

package main

import "fmt"

func main() {
    fmt.Println("go" + "lang")
    fmt.Println("1+1=", 1+1)
    fmt.Println("7.0/3.0=", 7.0/3.0)
    fmt.Println(true && false)
    fmt.Println(true || false)
    fmt.Println(!true)
}

Strings 可以通过＋来拼接。 Intergers，floats，booleans等都和我们之前所学的语言类似。

$ go run values.go
golang
1+1= 2
7.0/3.0= 2.3333333333333335
false
true
false
